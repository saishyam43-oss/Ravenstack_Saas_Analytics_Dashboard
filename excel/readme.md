# Excel Files Overview

This folder contains Excel workbooks used for **data validation and summary reporting** during the data cleaning phase of the project.

---

## Files

- **data_cleaning_steps.xlsx**  
  Documents the step-by-step data cleaning and validation process.

- **data_cleaning_summary.xlsx**  
  Summarizes data quality metrics and forms the basis of the data cleaning summary dashboard.

---

## Usage Notes
- Excel was used intentionally for validation and auditability.
- No manual data modifications were performed on final analytical datasets.
- Final analysis and feature engineering were completed using Python.
